oklch(0.8 0.12 100)
oklch(0.8 0.12 200)
oklch(0.8 0.12 300)
oklch(.95 0 300)

First number is brightness 0-1, 1 is bright. Second number is saturation 0-1, 0 is no saturation. Third number is color 0-300

0=magenta

50=red (13-88 seems to be red at full saturation)

95=orange

110=yellow

150=green

200=cyan

250=blue

300=purple

(Then numbers seem to rotate again)

Transparency is oklch(1 1 300 / .5)
